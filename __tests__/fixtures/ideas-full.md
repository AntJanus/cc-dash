## Project requirements

At minimum, a project should start with:

- a folder
- a README.md in the folder describing the project
- a ROADMAP.md for tracking progress

## Status Legend

- `NOT STARTED` -- Idea only, no repo exists
- `STARTED` -- Repo scaffolded, work in progress
- `COMPLETE` -- Shipped / feature-complete
